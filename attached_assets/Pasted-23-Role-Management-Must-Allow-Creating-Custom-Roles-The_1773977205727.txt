26. Role Permission Should Work for All Sidebar Menus

Permissions should control visibility and access for all sidebar menus and their pages.

If a user does not have permission for a module, that menu should be hidden or restricted.

If they have view only, they can open it but cannot add/edit/delete.

Example Logic

User with permission for Performance Reports can see and open that page

User without permission for Settings should not see Settings in sidebar

User with only View permission for Students Directory can open and see records but cannot edit
