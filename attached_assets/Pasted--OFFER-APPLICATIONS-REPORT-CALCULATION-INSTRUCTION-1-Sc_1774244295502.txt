📊 OFFER APPLICATIONS – REPORT CALCULATION INSTRUCTION
1. Scope

This report is for Offer / Admission Applications only.

Used in:

Workload Overview
Staff Handling Time
Stage Analysis
Offer Total Number
Avg Handling Time
Avg Completion Time
2. OFFER WORKFLOW (STANDARD)
Enquiries → (Review)

→ Document Requested (if missing docs)
→ On Hold (if incomplete / waiting)
→ Not Eligible (if not qualified)

→ Offer Requested
   → Offer Received ✅
   → Offer Rejected ❌
3. STATUS DEFINITIONS
🟡 Active / In-Progress

These are ongoing cases:

Enquiries
Document Requested
On Hold
🔵 Processing Stage

Core admission processing:

Offer Requested
✅ Completed (Final Outcome)
Offer Received (Positive)
Offer Rejected (Negative)
Not Eligible (Closed without processing)
❗ 4. CURRENT ISSUE

Right now:

Final statuses (Offer Received / Rejected / Not Eligible)
are treated as completed only
❌ but NOT included in:
Offer total
Workload
Handling time

👉 This causes incorrect reporting

Enquiries

+ Offer Requested
+ Offer Received
+ Offer Rejected
+ Not Eligible

✅ 6. WORKLOAD CALCULATION
Definition:

All applications handled by staff (active + completed)

Include:
Enquiries
Document Requested
On Hold
Offer Requested
Offer Received
Offer Rejected
Not Eligible
Formula:
Workload = Count of all Offer applications assigned to staff

⚠️ IMPORTANT:
Even if application is completed → MUST be included in workload

✅ 7. COMPLETION COUNT
Completed Offer Cases:
Offer Received
Offer Rejected
Not Eligible
Formula:
Completed Cases =
Offer Received + Offer Rejected + Not Eligible
✅ 8. AVG HANDLING TIME
Definition:

Time taken to handle an application (from assignment to latest status)

Include ALL statuses:
Enquiries
Document Requested
On Hold
Offer Requested
Offer Received
Offer Rejected
Not Eligible
Formula:
Handling Time per Case =
Latest Status Date - Assigned Date

Avg Handling Time =
Total Handling Time / Total Cases

⚠️ Must include completed cases

✅ 9. AVG COMPLETION TIME
Only for completed cases:
Offer Received
Offer Rejected
Not Eligible
Formula:
Completion Time per Case =
Final Status Date - Assigned Date

Avg Completion Time =
Total Completion Time / Completed Cases
